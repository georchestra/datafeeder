# Shared Components

This folder contains reusable UI components that can be used across multiple features of the application.

## Purpose

Shared components are generic, presentational components that:
- Have no business logic or feature-specific knowledge
- Are highly reusable and configurable through inputs and outputs
- Focus on presentation and user interaction
- Can be used in any feature module

## Examples

Common shared components include:
- Buttons and form controls
- Cards and containers
- Modals and dialogs
- Loading spinners
- Data tables
- Alert/notification components
