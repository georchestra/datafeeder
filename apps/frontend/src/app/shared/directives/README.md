# Shared Directives

This folder contains custom Angular directives that extend HTML element behavior and can be reused across the application.

## Purpose

Directives provide reusable DOM manipulation and behavior logic:
- Attribute directives that modify element appearance or behavior
- Structural directives that change DOM layout
- Common UI interactions and enhancements
- Accessibility improvements

## Examples

Common shared directives include:
- Click outside detection
- Tooltip functionality
- Auto-focus controls
- Permission-based visibility
- Responsive behavior helpers
