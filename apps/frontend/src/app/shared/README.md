# Shared Module

This folder contains reusable components, directives, pipes, and utilities that are shared across multiple features. Items here should be generic and have no feature-specific logic.

## Contents

- **components/** - Reusable UI components (buttons, cards, modals, etc.)
- **directives/** - Custom Angular directives for DOM manipulation
- **pipes/** - Custom Angular pipes for data transformation

## Guidelines

- Keep components generic and configurable through inputs
- Avoid feature-specific logic
- Document component APIs clearly
- Components here can be imported by any feature module
