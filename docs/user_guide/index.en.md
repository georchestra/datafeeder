# User guide

Datafeeder is used by platform administrators and data managers to bring a dataset into geOrchestra and to keep it
published over time. This guide follows a dataset through its whole lifecycle:

- **[Importing a dataset](importing_data.md)**: pick a source, stage it, configure the transformation and publish it.
- **[Managing datasets](managing_datasets.md)**: the dashboard listing every dataset, its status, its recurring
  refresh schedule and how to delete it.
- **[Editing metadata](metadata.md)**: fill in the GeoNetwork record attached to a dataset.
- **[Authorizations](authorizations.md)**: control who can see and edit which datasets and metadata.

The application is reachable through the geOrchestra gateway, which handles authentication: `/datafeeder/`.
